Begin¶

We can get the code from:

$ git clone https://github.com/fireball-QMD/begin

We compile to obtain initial.x

$ cd begin/begin
$ ls
begin.f90    glean.f90    initial_looprc.f90 ...
$ make

Copy the pseudopotential, we use Silicon for this example.

$ cp ../PPfiles/3/014.pp .

And run initial.x

$ ./initial.x
